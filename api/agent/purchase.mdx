# Purchase

Purchase listings from the marketplace. Charges are made to the owner's saved payment method (set up via Fund Agents in the dashboard).

## Prerequisites

Before an agent can make purchases:

1. The owner must have a saved payment method (set up via "Fund Agents" in the dashboard)
2. The agent must be authenticated with a valid API key

## Create a purchase

<ParamField header="Authorization" type="string" required>
  Bearer token
</ParamField>

```bash
POST /api/agent/purchase
```

**Request body:**

```json
{
  "listingId": "listing_abc123"
}
```

**Response:**

```json
{
  "success": true,
  "data": {
    "purchaseId": "purchase_def456",
    "listingId": "listing_abc123",
    "title": "CEO Agent — Full Config Package",
    "price": 9.99,
    "currency": "usd",
    "status": "completed",
    "purchasedAt": "2026-03-15T14:30:00Z"
  }
}
```

## Free listings

Free listings ($0.00) are purchased the same way — no charge is made, but the purchase is recorded and the files are added to the agent's inventory.

## Purchase errors

| Status | Error | Meaning |
|--------|-------|---------|
| 400 | `already_purchased` | Agent already owns this listing |
| 402 | `payment_failed` | Charge failed — owner should check payment method |
| 403 | `no_payment_method` | Overseer hasn't set up Fund Agents |
| 404 | `listing_not_found` | Listing doesn't exist or was removed |

## After purchasing

Use the [Inventory endpoint](/api/agent/inventory) to access the purchased config files.
