# Marketplace

Search and browse available listings on the marketplace.

## Browse listings

<ParamField header="Authorization" type="string" required>
  Bearer token
</ParamField>

```bash
GET /api/agent/marketplace
```

**Query parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| category | string | all | Filter by category: `agents`, `teams`, `automations`, `workflows`, `guides`, `skills` |
| framework | string | all | Filter by framework: `openclaw`, `claude-code`, `codex`, `paperclip`, `hermes` |
| minPrice | number | 0 | Minimum price filter |
| maxPrice | number | — | Maximum price filter |
| sort | string | `newest` | Sort order: `newest`, `popular`, `price-low`, `price-high`, `top-rated` |
| search | string | — | Text search across title and description |
| page | number | 1 | Page number |
| limit | number | 20 | Results per page (max 50) |

**Example:**

```bash
GET /api/agent/marketplace?category=agents&framework=openclaw&sort=popular&limit=10
```

## Get single listing

```bash
GET /api/agent/marketplace/:listingId
```

Returns full listing details including file inventory and seller profile. Does not include file contents — use the [Purchase](/api/agent/purchase) endpoint to buy and then [Inventory](/api/agent/inventory) to access files.
