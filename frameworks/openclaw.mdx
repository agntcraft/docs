# OpenClaw Setup

OpenClaw is the most popular open-source agent framework with 307K+ GitHub stars. It reads markdown config files from your agent's character directory.

## How wizard files integrate

The AgentCraft wizard generates 7 universal UPPERCASE files that complement OpenClaw's own config. Place them in the character directory alongside OpenClaw's native files (like `character.json`). The wizard does not generate framework-specific files — it generates universal files that any framework can read.

| Wizard file | Where it goes |
|-------------|---------------|
| README.md | Character directory |
| SOUL.md | Character directory |
| AGENTS.md | Character directory |
| USER.md | Character directory |
| HEARTBEAT.md | Character directory |
| IDENTITY.md | Character directory |
| TOOLS.md | Character directory |

## Quick start

1. [Generate a config](https://agntcraft.co/create) — select "OpenClaw" as framework
2. Download the ZIP
3. Extract into your OpenClaw agent's character directory alongside your existing `character.json`
4. Start your agent — it reads the configs automatically

## Character directory structure

```
my-agent/
  character.json    ← OpenClaw's native config (you write this)
  README.md         ← Wizard output — agent reads this first
  SOUL.md
  AGENTS.md
  USER.md
  HEARTBEAT.md
  IDENTITY.md
  TOOLS.md
```

The wizard files sit alongside OpenClaw's own `character.json` — they complement it, not replace it.
