# OpenClaw Setup

OpenClaw is the most popular open-source agent framework with 307K+ GitHub stars. It reads soul.md and other markdown config files from your agent's character directory.

## Config mapping

| AgentCraft file | OpenClaw equivalent | Where it goes |
|-----------------|-------------------|---------------|
| soul.md | soul.md (native) | Character directory |
| agents.md | agents.md | Character directory |
| heartbeat.md | heartbeat.md | Character directory |
| identity.md | identity.md | Character directory |
| tools.md | tools.md | Character directory |
| character.json | Generated by wizard | Character directory root |

## Quick start

1. [Generate a config](https://agntcraft.co/create) — select "OpenClaw" as framework
2. Download the ZIP
3. Extract into your OpenClaw agent's character directory
4. Start your agent — it reads the configs automatically

## Character directory structure

```
my-agent/
  soul.md
  agents.md
  heartbeat.md
  identity.md
  tools.md
  character.json
```

## character.json

The wizard generates a `character.json` file that OpenClaw uses for model settings and identity metadata:

```json
{
  "name": "my-agent",
  "model": "claude-opus",
  "bio": "A brief description of the agent",
  "lore": [],
  "style": {
    "all": ["concise", "direct"],
    "chat": ["helpful"],
    "post": ["professional"]
  }
}
```

This file is auto-generated based on your wizard answers. The markdown files (soul.md, agents.md, etc.) provide the detailed instructions.
