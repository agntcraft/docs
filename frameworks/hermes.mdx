# Hermes Setup

Hermes is a lightweight agent framework that reads a `SOUL.md` file from `~/.hermes/` to define the agent's personality and behavior. It supports personality overlays for context-specific adjustments.

## How wizard files integrate

The AgentCraft wizard generates 7 universal UPPERCASE files that complement Hermes' own `SOUL.md`. The wizard does not replace Hermes' native SOUL.md — the wizard files sit alongside it, providing additional structure for identity, operating instructions, and tools.

| Wizard file | Where it goes |
|-------------|---------------|
| README.md | `~/.hermes/` |
| SOUL.md | `~/.hermes/` |
| AGENTS.md | `~/.hermes/` |
| USER.md | `~/.hermes/` |
| HEARTBEAT.md | `~/.hermes/` |
| IDENTITY.md | `~/.hermes/` |
| TOOLS.md | `~/.hermes/` |

## Quick start

1. [Generate a config](https://agntcraft.co/create) — select "Hermes" as framework
2. Download the ZIP
3. Extract the wizard files into `~/.hermes/`
4. Start Hermes — it reads the files automatically

## File location

```
~/.hermes/
  README.md        ← Wizard output — agent reads this first
  SOUL.md
  AGENTS.md
  USER.md
  HEARTBEAT.md
  IDENTITY.md
  TOOLS.md
  overlays/        ← Optional Hermes personality overlays
    coding.md
    writing.md
```

## Personality overlays

Hermes supports overlays — additional markdown files that modify the base persona for specific contexts. For example, a `coding.md` overlay might add stricter formatting rules when the agent is writing code, while a `writing.md` overlay might shift to a more creative tone.

Overlays don't replace SOUL.md — they layer on top of it.
