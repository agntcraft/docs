# AGENTS.md — Codex Instructions

AGENTS.md is the instruction file that OpenAI Codex reads before doing any work. It's standard markdown — no special syntax needed. Codex layers global guidance with project-specific overrides.

<Info>
  **Generate a custom AGENTS.md** — Use the [AgentCraft Wizard](https://agntcraft.co/create) and select "Codex" as your framework.
</Info>

## What goes in AGENTS.md

- **Project structure** — how the codebase is organized
- **Architecture** — key patterns and decisions
- **Coding standards** — style and conventions
- **Agent behavior** — how Codex should approach tasks
- **Rules** — boundaries and constraints

## Template

```markdown
# AGENTS.md

## Project Overview
[What this project does, tech stack, key dependencies]

## Structure
[Directory layout, key files, module organization]

## Standards
[Coding conventions, naming patterns, testing requirements]

## Behavior
[How the agent should approach tasks, make decisions, communicate]

## Rules
[What to always/never do, when to ask for help]
```

## Where Codex looks for AGENTS.md

Codex reads files in this order:
1. `~/.codex/AGENTS.md` (global)
2. Project root `AGENTS.md`
3. Current directory `AGENTS.md`

Files are layered — project-level overrides global settings.
