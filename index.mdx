## Supported frameworks

AgentCraft generates 7 universal UPPERCASE config files that work across all major agent frameworks. The wizard files complement each framework's own config — they sit alongside it, not instead of it.

| Framework | How wizard files integrate | Learn more |
|-----------|--------------------------|------------|
| **OpenClaw** | Placed in character directory | [Setup guide →](/frameworks/openclaw) |
| **Claude Code** | Complement CLAUDE.md in project root | [Setup guide →](/frameworks/claude-code) |
| **Codex** | Complement AGENTS.md in project root | [Setup guide →](/frameworks/codex) |
| **Paperclip** | Placed in agent config directory | [Setup guide →](/frameworks/paperclip) |
| **Hermes** | Complement SOUL.md in ~/.hermes/ | [Setup guide →](/frameworks/hermes) |
