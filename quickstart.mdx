# Quickstart

Three ways to get a production-ready agent configuration.

## Option 1: The Wizard (recommended)

The fastest way to get started. No code, no terminal.

1. Go to [agntcraft.co/create](https://agntcraft.co/create)
2. Answer 8 questions about your agent (name, role, personality, tasks, framework)
3. Preview the generated config files
4. Download the full package ($9.99)

The wizard generates: `soul.md`, `agents.md`, `heartbeat.md`, `identity.md`, `tools.md`, plus a framework-specific config file.

## Option 2: The Marketplace

Browse configs built by experienced sellers.

1. Go to [agntcraft.co/browse](https://agntcraft.co/browse)
2. Filter by category, framework, or price
3. Preview the file inventory before buying
4. Download as a ZIP after purchase

## Option 3: The CLI (coming soon)

Generate configs from your terminal. Free, open source, zero dependencies.

```bash
npx agntcraft init
```

Interactive prompts guide you through the same questions as the wizard. Files are generated locally in your project folder.

## After downloading

Drop the config files into your agent framework's config directory:

| Framework | Where to put files |
|-----------|-------------------|
| OpenClaw | Your agent's character directory |
| Claude Code | Project root (`CLAUDE.md`) |
| Codex | Project root (`AGENTS.md`) or `~/.codex/` |
| Paperclip | Project scaffold config directory |
| Hermes | `~/.hermes/SOUL.md` |

See the [framework guides](/frameworks/openclaw) for detailed setup instructions.
